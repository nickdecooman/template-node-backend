## Features
* Docker driven
* Build and run development image with auto-restart
* Build and run stripped production image
* Uses Koa, Babel, Flow, ESLint and Prettier
* Support for ES6 modules
* Jest for unit testing 

## Usage

### Development

```
yarn run build-dev
yarn run up-dev
``` 

### Production

```
yarn run build-prod
yarn run up-prod
```
